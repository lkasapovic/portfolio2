Hypertext Markup Language, a standardized system for tagging

Bootstrap

Bootstrap is the most popular HTML, CSS, and JS framework for
developing responsive, mobile first projects on the web.

Javascript

An object-oriented computer programming language commonly used

jQuery

jQuery is a cross-platform JavaScript library designed to simplify
the client-side scripting of HTML. Used by over 60% of the 10,000 most
visited websites, jQuery is the most popular JavaScript library in
use today. jQuery is free, open-source software licensed under the MIT License.
